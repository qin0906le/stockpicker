# 🔮 紫微斗数排盘分析

纯静态、零后端的紫微斗数排盘与解读工具。在浏览器中打开 `index.html` 即可使用。

## 功能

输入以下资料：

- **基本资料**：姓名（可选）、性别、公历出生日期、出生时间、出生地点（可选）
- **目前状况（选填）**：职业／工作状况、婚姻状态、居住国家／城市

即可自动生成：

1. **命盘基本信息** — 农历生日、四柱、五行局、命主／身主、命宫／身宫、生肖星座、当前虚岁
2. **十二宫命盘** — 经典方盘布局：十四主星（含庙旺利陷亮度）、六吉六煞、禄存天马、杂曜、
   生年四化标记、长生十二神、各宫大限年龄段；空宫自动借对宫主星；标注身宫、当前大限与流年所在宫位
3. **命盘解读** — 命宫总论（十四主星详解）、五行局／命主身主／身宫、
   重点宫位（财帛・官禄・夫妻・迁移・福德）解读、其余宫位速览、生年四化总览
4. **大限与流年** — 当前十年大限宫位与大限四化、本年度流年宫位与流年四化
5. **结合目前状况** — 根据填写的职业、婚姻、居住地，针对性给出官禄宫、夫妻宫、迁移宫的解读

## 使用方式

```bash
# 任选一种方式启动本地服务（也可直接双击 index.html 打开）
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000/ziwei/
```

## 技术说明

- 排盘核心使用 [iztro](https://github.com/SylarLong/iztro)（v2.5.8，已打包内置 `iztro.min.js`），
  基于三合派规则排盘，支持 1900–2100 年农历换算、闰月、早晚子时
- 解读文本库见 `interpretations.js`：十四主星×重点宫位、四化、六吉六煞、五行局等
- 无任何构建步骤与外部网络依赖，纯 HTML/CSS/JS
